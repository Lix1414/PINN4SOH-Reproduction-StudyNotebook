## ZIP 解压后验证路径

从仓库根目录依次执行：

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python scripts/check_setup.py --full
python src/02_dataloader.py
python src/03_model.py
python src/04_loss.py
python src/verify_pipeline.py
python scripts/run_minimal_training.py
```

本次验证使用环境变量将 `PINN4SOH_DATA_ROOT` 指向本地数据目录；使用默认目录时，应将 CSV 放在 `data/XJTU data/`。
